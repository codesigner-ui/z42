# toolchain/workload — 端到端工作流与发行场景

## 职责

端到端工作流的验证与发行场景：真实规模的 z42 应用样例、打包-分发-运行全链路测试、性能基准。

与 `examples/` 的区别：后者是语言特性的最小示例；本模块是完整可发布的示范应用 + 流水线验证。

与 `src/compiler/z42.Tests/` / `scripts/test-*` 的区别：测试覆盖编译器与 VM 正确性；本模块覆盖 **从源码到最终用户运行** 的完整链路。

不做：单元测试（归各自模块内）。

## 计划模块

尚未实现。预期包含：

| 模块 | 职责 |
