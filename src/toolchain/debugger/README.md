# toolchain/debugger — z42 调试器

## 职责

z42 程序的调试工具：断点、单步、调用栈、变量查看。基于 VM 的调试钩子与 zbc 中的源位置元数据（`z42ir.json` 的 line/col 信息）。

不做：语言服务器 / LSP（独立能力，未来可能放到 toolchain 下同级）。

## 计划模块

尚未实现。预期包含：

| 模块 | 职责 |
|------|------|
| `protocol/` | 调试适配协议（DAP 或自定义） |
| `engine/` | 调试引擎（对接 VM 的断点/单步钩子） |
| `cli/` | 命令行调试器前端 |

## 依赖关系

- 依赖 `runtime/`（调试钩子）、`compiler/z42.IR`（读取 zbc 元数据）
- 被 IDE / 命令行调用
