# z42.collections — 集合库

## 职责

z42 标准集合类型。

## src/ 核心文件

| 文件 | 类型 | 说明 |
|------|------|------|
| `List.z42` | `List<T>` | 动态数组 |
| `Dictionary.z42` | `Dictionary<K,V>` | 哈希映射 |
| `HashSet.z42` | `HashSet<T>` | 哈希集合 |
| `Queue.z42` | `Queue<T>` | 队列（FIFO） |
| `Stack.z42` | `Stack<T>` | 栈（LIFO） |
